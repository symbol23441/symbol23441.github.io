几种线程的创建方式

本文将展示几种主要的Java线程的创建方式：

  • 继承Thread
  • 实现Runnable接口（推荐）
  • 实现Callable接口+Future（带线程返回值）
  • Lamda 表达式创建
  • 自定义线程池创建

这些线程的创建本质上是实现了Runnable接口。

1、继承 Thread 类

适合简单任务，但不推荐继承 Thread 和实现 Runnable 同时使用，因为 Java 不支持多继承。

2、实现 Runnable 接口（推荐）

优点：更灵活，可以避免继承限制；可用于多个线程共享资源。

3、实现 Callable 接口 + FutureTask（有返回值）

优点：可以有返回值，且支持异常处理。

4、lambda表达式创建（Java 8 及以上）

语法简洁，常用于简短任务。

5、线程池创建

实际中最多使用的是自定义线程池ThreadPoolExcutor。避免使用Java提供的Excutor因为很多因素不可控。

适合高并发或大量任务执行场景，推荐在线程数量受控的情况下使用。
